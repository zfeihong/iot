FineUI JS (F.js)

基于 jQuery 的专业 WebUI 控件库

支持的浏览器

Chrome、Firefox、Safari、Edge、IE8.0+*

  • 论坛：http://fineui.com/bbs/

* 推荐使用 Chrome、Firefox、Safari、Edge、IE11 浏览器，以获取最好的性能。
IE8.0 浏览器有限支持，存在如下已知问题：
  • 相比其他现代浏览器，性能最差
  • 不支持圆角，以及其他一些 CSS3 特性
  • 选项卡控件不支持左侧和右侧标题栏
  • IFrame 交互时，可能会随机出现字体图标丢失的问题
